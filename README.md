### SQL 
The Project focused on working with Relational Database from Python using SQLAlchemy.
Database was created in PostgreSQL.
Basic Analysis was done on Employees Salaries.

#### Libraries used:
* Sqlalchemy, Pandas, Matplotlib.
#### Database:
* PostgreSQL.
